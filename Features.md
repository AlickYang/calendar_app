# Calender with Todo List
Built on MERN stack with GraphQL, based off code_realm

## Features
### MVP 
- Todo list connected to each day within calendar
- Responsive material design

### Extra Features
- Notifications
- Themes and customisation
- Other

### Topics:
- Schema Design
- User validation
- Authentication (session vs token)
- Authorisation (access control)
- Security (injections, XSS, CSRF, headers)
- State management
- File upload
- Real-time updates
- Performance optimsation
- Pagination

### Technologies
#### Backend: 
- Node + Express
- MongoDB + Mongoose ORM
- Apollo Server v2
- Web sockets
- Redis
#### Front-End
- React 
- Apollo Client
- Material-UI
- Bootstrap + Reactstrap
